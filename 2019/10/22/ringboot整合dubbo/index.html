springboot整合dubbo

2019-10-22

Apache Dubbo |ˈdʌbəʊ| 是一款高性能、轻量级的开源Java RPC框架，它提供了三大核心能力：面向接口的远程方法调用，智能容错和负载均衡，以及服务自动注册和发现。最初由阿里巴巴研发，现已捐献给apache

官方文档：http://dubbo.apache.org/zh-cn/

Dubbo一般使用zookeeper来当注册中心，所以在使用dubbo之前先安装zookeeper

Dubbo采用全spring配置方式，基于 Spring 的 Schema 扩展 进行加载。如果不想使用 Spring 配置，可以通过 API 的方式 进行调用。

但是使用spring的方式还是需要大量的xml配置，比较繁琐，所以这里记录一下使用springboot整合Dubbo
